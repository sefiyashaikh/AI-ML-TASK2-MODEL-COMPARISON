# AI/ML Task-2: Feature Engineering & Model Comparison

## 📌 Project Overview
This project applies multiple regression models to predict housing prices using the California Housing dataset.

## 🚀 Models Used
- Linear Regression
- Ridge Regression
- Decision Tree Regressor

## 📊 Evaluation Metrics
- RMSE (Root Mean Squared Error)
- R² Score

## 📈 Key Learnings
- Feature scaling improves linear model performance.
- Ridge regression reduces overfitting.
- Model comparison helps select best performing algorithm.

## 🛠 Tools Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
